Computer Science 111

Introduction to C++

CS 111

CS111(Introduction to Algorithmic Problem- Solving): Introductory course for computer science department. Learn the principles and practice of programming using C++. Topics include primitive data types; control structures; arrays; procedures and functions; parameter passing; scope and lifetime of variables; input and output; documentation.

Computer Science 211

Intermediate C++

CS 211

CS211(Object-Oriented Programming in C++): Intermediate programming with C++. Object-oriented algorithmic problem solving in C++, with attention to general as well as language-specific issues including pointer and pointer arithmetic; linked lists; memory management; recursion; operator overloading;emplates and STL; applications of simple data structures; testing and debugging techniques.

Java Language

Object Oriented Programming in Java

Java

Object-oriented algorithmic problem solving in Java. Learn the principles and practice of programming using java. event-driven programming; elements of graphical user interfaces (GUIs); recursion; inheritance and polymorphism; file I/O; exception handling; packages; applications of simple data structures;concept of multithreading.

Data Structures

Data Structures in Java

DS

Fundamental data structures and their implementations: list, stacks, queues, trees (binary and AVL), heaps, hash tables. Searching and sorting algorithms. Runtime analysis.
